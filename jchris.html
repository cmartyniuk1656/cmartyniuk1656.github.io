GENERAL INFORMATION

jChris is a small personal library created for educational purposes. Developing jChris was a study in HTML, CSS, media queries, animations, transitions, JSON, AJAX, JavaScript, JavaScript object inheritence prototyping, cross browser compatibility techniques, JavaScript function chaining, and an in-depth look at how frameworks like jQuery and BootStrap operate under the hood.

WHAT IS IT

jChris contains a small number of helpful functions that help to code more efficiently, a small CSS library of classes that help style and structure responsive webpages with rows, columns, and media queries, as well as a few data-attribute widgets that automate functionality for notated HTML elements.


USES

Simplifies HTML Node Selection / Manipulation

Abbreviates JavaScript syntax involved with getting and manipulating HTML Node objects (like jQuery.)

Function Chaining and multi-selector strings

Execute multiple functions on multiple selectors in a single call that is easy and efficient to write, and easy to read.

AJAX Internal Domain HTML Node Retrieval

Grab any HTML Node object from any file on the domain by it's ID and inject it into the DOM with quick and easy jChris AJAX functions.

Preset Media Queries and Breakpoints

Use the jChris column and row classes to structure your website and it will automatically resize and rearrange it's anatomy based on the client screen display size.


jChris SELECTORS

Similar to jQuery, jChris can select and manipulate HTML node objects with simplified selector query notation. Strings, nodes, node lists, HTML objects, and HTML object collections can all be passed into the jC() object as selectors.


STRING SELECTORS

Description

Strings can be used to identify desired selector types using the standard selector notation used in jQuery and querySelector functions. The following notators can be used: "#" is used for specifying ID, '.' for class types, "<>" for tags, and '[]' for indices if multiple nodes are queried.

Examples

jC('#myId') : Selects a node with an ID of 'myId'.

jC('.myClass') : Creates a node list of every node in the DOM with a class of 'myClass'.

jC('.myClass[0]') : Selects the first node in the DOM with a class of 'myClass'.

jC('<p>') : Creates a node list of every node in the DOM with the 'p' tag.

jC('<p>[0]') : Selects the first node in the DOM with a 'p' tag.

STRING MULTI SELECTORS

Description

Multiple selectors can be combined in one call with string selectors seperated by commmas. This can be used to execute the same functions on multiple nodes efficiently.

Examples

jC('#firstSelector, #secondSelector') : Creates a node list containing 2 nodes with an ID of 'firstSelector' and 'secondSelector'.

jC('#firstSelector, .green, <h1>[1]') : Creates a node list containing a node with an ID of 'firstSelector', all nodes with a CSS class of 'green', and the second h1 node found int he DOM.


jChris FUNCTION CHAINING

Similar to jQuery, jChris supports function chaining using inheritance prototyping by passing itself back into the original jChris constructor after each chained operation. This can increase efficiency greatly by reducing the amount of effort needed to execute multiple functions on the same node/nodes, even more so when combined with string multi-selectors.


CHAINED FUNCTION EXAMPLES

Chaining Functions on a Single Node

jC('#pageContent').showElement().addClass('green') : Show the node with an ID of 'pageContent' if it is hidden then add a CSS class called 'green' to it if it doesnt already have it.

Chaining Functions on multiple Nodes

jC('.navBtn, .selectedContent').removeClass('selected')removeClass('selectedContent') : Removes CSS classes(if the nodes posses them) named 'selected, and 'selectedContent' from all nodes containing the CSS classes 'navBtn' and 'selectedContent'.


jChris FLOWCHART



.ret()

Description

Returns an HTML Node object or Node list from the selector string passed into the the jC() object in the original set.

Example

jC('#myId').ret();

Returns and HTML Node object with the ID of 'myId'


.defined()

Description

Evaluates the value of a javascript object. If the value is of type "undefined", no further chained functions will be executed, and a value of false is returned. If the object is evaluated and the value is not "undefined", chained function execution continues.

Example

jC('#myId').defined().chainedFunction();

If an HTML Node witht he ID of 'myId' exists in the current DOM, subsequent chained function calls will execute.


.addEvent(type [string], func [function])

Description

Adds an event handler to the specified node / nodes provided by the jC() selector argument.

Parameters

Type [string] : Define the type of event to listen for.

Func [function] : Define the function to execute when the listener is triggered.

Example

jC('#myId').addEvent('click', function( { this.style.color = 'green' }));

When the HTML Node with an ID of 'myId' is clicked, its CSS 'color' style will become 'green'.


.removeEvent(type [string], func [function])

Description

Removes an event handler from the specified node / nodes provided by the jC() selector argument.

Parameters

Type [string] : Define the type of event that will be removed.

Func [function] : Identify the function that should be removed from the event listner. Does not work on annnymous functions.

Example

jC('#myId').removeEvent('click', definedFunction));

When the HTML Node with an ID of 'myId' is clicked, the previously defined and added event listener function 'definedFunction()' will no longer execute.


.click(func [function])

Description

Adds an event for a click listener on the node / nodes selected from the original jC() string selector set.

Parameters

Func [function] : This function will execute when selected node / nodes are clicked.

Example

jC('#myId').click( function() { jC(this).hide() };);

When the node '#myId' is clicked it will be hidden from view.


.mouseover(func [function])

Description

Adds an event for a mouse over listener on the node / nodes selected from the original jC() string selector set.

Parameters

Func [function] : This function will execute when selected node / nodes are moused over.

Example

jC('#myId').click( function() { jC(this).hide() };);

When the node '#myId' is moused over it will be hidden from view.


.mouseout(func [function])

Description

Adds an event for a mouse out listener on the node / nodes selected from the original jC() string selector set.

Parameters

Func [function] : This function will execute when selected node / nodes are moused out. (mouse over ends)

Example

jC('#myId').click( function() { jC(this).hide() };);

When the cursor is no longer moused over tje node '#myId' it will be hidden from view.


.addClass(className [String])

Description

Adds a CSS class on the node / nodes selected from the original jC() string selector set.

Parameters

ClassName [string] : String name of the CSS class that will be added.

Example

jC('#myId').addClass('myClass');

A class with the name of 'myClass' will be added to a node witht an ID of '#myId'.


.removeClass(className [String])

Description

Removes a CSS class on the node / nodes selected from the original jC() string selector set.

Parameters

ClassName [string] : String name of the CSS class that will be removed.

Example

jC('#myId').removeClass('myClass');

A class with the name of 'myClass' will be removed from a node witht an ID of '#myId'.


.hasClass(className [String])

Description

Evaluates the existance of a CSS class on the selected HTML node. If the node does not have the specified class, chained functions will not be executed, and a value of false is returned. If the class is found, chained function execution continues.

Parameters

ClassName [string] : String name of the CSS class that will be evaluated.

Example

jC('#myId').hasClass('myClass').chainedFunction();

If an HTML Node in the current DOM with he ID of 'myId' has a CSS class with the name of 'myClass', subsequent chained function calls will execute.


.setAttribute(attribute [String], value [string])

Description

Set an HTML attribute value for the selected HTML Node / Nodes.

Parameters

Attribute [string] : String name of the HTML attribute that will be set.

Value [string] : Value to set the attribute to as a string.


.pointer()

Description

The mouse will become a pointer cursor when mousing over the selected element/elements.

Parameters

None

Example

jC('.clickableLink').pointer();

All nodes with the CSS class 'clickableLink' will make the cursor into a pointer hand when moused over.


.fadeIn()

Description

Adds a fade in animation class to selected elements.

Parameters

None

Example

jC('#newElement').fadeIn();

Fades in element with an ID of 'newElement'.


.fadeOut()

Description

Adds a fade out animation class to selected elements.

Parameters

None

Example

jC('#newElement').fadeOut();

Fades out element with an ID of 'newElement'.


.halfFadeTransition()

Description

Adds a fade out transition class to selected elements. Reduces opacity to 50%.

Parameters

None

Required

Element must have the fullOpacity jChris CSS class.

Example

jC('#newElement').halfFadeTransition();

Fades out element with an ID of 'newElement' to 50% opacity.


.fadeInTrans()

Description

Adds a fade in transition class to selected elements. Sets opacity to 100%.

Parameters

None

Required

Element must have the fadeHalf or noOpacity jChris CSS class.

Example

jC('#newElement').fadeInTrans();

Fades in an element with an ID of 'newElement' to 100% opacity.


.hide()

Description

Hides an element from view in the browser. Sets display attribute to 'none'.

Parameters

None

Example

jC('#hiddenElement').hide();

Element with an ID of 'hiddenElement' can no longer be viewed by the user.


.showElement()

Description

Reveals a hidden element. Sets display attribute to 'block'.

Required

Element must have display attribute set to 'none'.

Parameters

None

Example

jC('#newelement').showElement();

Element with an ID of 'newElement' can now be seen be viewed by the user.


.backgroundImg(url [string])

Description

Changes the background attribute of an element.

Parameters

URL: This string defines the url of the new background image.

Example

jC('#background').backgroundImg("url('images/newBackground.jpg')");

Element with an ID of 'background' now has a background image attribute url of 'images/newBackground.jpg'.
